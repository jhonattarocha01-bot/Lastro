# Big Wave Inspired - versão pronta para Vercel

## Estrutura
- `index.html`
- `styles.css`
- `script.js`
- `vercel.json`

## Como publicar na Vercel
1. Crie um repositório no GitHub.
2. Envie esses arquivos para o repositório.
3. Entre na Vercel.
4. Clique em **Add New > Project**.
5. Importe o repositório.
6. Clique em **Deploy**.

Como é um site estático, a Vercel publica sem configuração extra.

## O que editar
- Textos: edite diretamente no `index.html`
- Cores e layout: edite no `styles.css`
- Ações do menu e formulário: edite no `script.js`

## Observação
Este projeto foi inspirado na estrutura do site de referência, mas com implementação original e sem copiar assets proprietários.
